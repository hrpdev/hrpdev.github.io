Hugoの設定とgithub.pagesの公開

1.hugoの設定 hugoとは go言語製サイトジェネレーター。 homebrewのインストール まずはhomebrewのインストールが必要。既に入ってるいる人は飛ばしてください。homebrewとはMacにソフトウェアやライブラリをインストールするパッケージ管理システム。homebrewサイト内にある以下のテキストをterminalにペースト。 /usr/bin/ruby -e "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/master/install)" hugoのインストール $ brew update $ brew install hugo ついでにbrewをアップデート 新規サイト構築 下記のコマンドを入力すると雛形が生成されます。 $ hugo new site my-site(お好きなサイト名) my-site |-config.toml |-archetypes |-content |-data |-layouts |-public |-static |-themes ディレクトリ内を軽く説明 ・archetypesは各ジャンルの雛形を入れるフォルダです。サイト作成時にdefault.mdファイルが生成されて書く記事に日付などをつけてくれるようになっています。 ・contentは各ジャンルの記事を入れておくフォルダです。 ・dataは様々なデータ入れて置くところです。（そのままです笑） ・layoutsはheaderやfotterなどの共通htmlファイルを入れて置く場所です。 ・publicはビルドされファイルの出力先です。 ・staticはcssや画像ファイルを入れて置く場所です。 ・themesはHugoのテーマを入れて置く場所です。 この状態ではプレビューしても何も出ませんのでひとまずテーマを設定します。 テーマの設定 ひとまずthemesフォルダに移動 以下のコマンドでおそらく全テーマがローカルにクローンされる $ git clone https://github.com/dim0627/hugo_theme_robust.git ひとつのテーマだけをインストールしたい人はhugo themesにお洒落なテーマが沢山あるのでテーマクリック→homepageクリック→gitのclone or downloadボタンにあるコードをコピペ $ git clone (コピペしたコードを貼り付け) インストールしたテーマを反映させるためにはconfig.tomlに下記を追加。テーマ名はthemesフォルダ直下にできたフォルダ名。 theme="テーマ名" これでserveすると選んだテーマが反映されます。 記事の作成 hugoの記事はマークダウン記法で書いていきます。 Read the post
